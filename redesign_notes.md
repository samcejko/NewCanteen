# iCanteen Redesign Notes

Tato aplikace prochází modernizací UI na Material Design 3.

## Klíčové změny
- Přechod na Neon Green / Dark mode schéma.
- Sjednocení navigace (Bottom Nav pro mobil, Sidebar pro desktop).
- Optimalizace parsování pomocí Java Bridge v Androidu.
- Odstranění závislostí na PHP/Python proxy pro produkční běh.

## Budoucí směřování
- Plná implementace Material You barev.
- Vylepšení stability parsování při změnách na straně serveru.
